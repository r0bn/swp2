## Setup

* NodeJS installieren https://nodejs.org/


Dependencies installieren
```
cd authorentool
npm install
```
* ggf. mit sudo ausführen

### Global Path Variable Problem

Sicherstellen das `node` in der Konsole ausführbar ist.

## Development Server

Starten
```
npm start
```
